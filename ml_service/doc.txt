Crear un entorno virtual en la raiz del proyecto de Django con:
python -m venv venv
en la raiz del proyecto activar el entorno virtual con el comando
venv\Scripts\activate

debe aparecer en la consola algo como:
(venv) C:\Users\...\ml_service>

una vez activado el entorno virtual instala las dependencias
pip install -r requirements.txt

Crea las migraciones
python manage.py makemigrations prediction

Aplica las migraciones
python manage.py migrate

Ejecuta el servicio 
python manage.py runserver
